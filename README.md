#make@os
simple setup for an angular app (with basic node/express serving files for routes)

#To run
`git clone git@github.com:sandagolcea/make-at-os.git`  
`cd make-at-os`  
`npm install`  
`bower install`  
`npm start`  

#Tests
- unit
`npm test`  

- e2e
`npm start`  
`npm run webdriver` - if error, run `npm run update-webdriver` first  
`npm run protractor`
